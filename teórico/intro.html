Qual a diferença entre Int main () ou Void main ()?

main() - função principal de um programa em C/C++. É onde o programa começa. Int - função que devolve no fim um número inteiro. Void - função que nada devolve, apenas se executa e chega ao fim sem retornar (return) nada. O int main(), pode ser terminado com um return 0 ou return 1. O void main(), esta função não devolve nada: quando esta chega ao fim, o programa chega ao fim também.
